**Instructions to Run AI Physician on a Different System**

### **Step 1: Install Required Libraries**

Open a terminal or command prompt in the project folder and run the following command to install dependencies:
pip install streamlit pillow google-generativeai

---
Step 2: Set Up API Key for Gemini
The application requires a Gemini AI API key. You need to:
1. Get an API key from [Google AI Studio](https://ai.google.dev/).

2.Change your api key ---->genai.configure(api_key="Your-Api-key")

Step 3 : Run the Application
streamlit run app.py
or
streamlit run ("the entire location where the file is placed")
Example: streamlit run "c:\Users\KIIT\Desktop\code\sem six\AiLab\sessional_2\app.py"
This will start the application in a web browser.

---
Step 4: Troubleshooting
- If `ModuleNotFoundError` occurs, ensure dependencies are installed (`pip install -r requirements.txt`).
- If API errors occur, double-check your API key.
- If Streamlit fails to start, restart your terminal and try again.

--

Now, the AI Physician should be up and running on the new system!
